This "res" diretory contains all code file necessary for the Android application interface.

In the "values" directory are all variables that will be called back in the whole application. It is used to describe names, or global variables.

In the "xml" directory are the interfaces monitoring of all application menus.

Some values and option interfaces were added and changed for the Dynamic Bolus Calculator
